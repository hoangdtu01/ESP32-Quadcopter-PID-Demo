# 🚁 ESP32 Quadcopter PID Demo

A mini quadcopter prototype built with ESP32, MPU6050, and PID control.

## Features

* ESP32-based flight controller
* MPU6050 DMP for attitude estimation
* Kalman filtering for roll and pitch smoothing
* PID stabilization control
* ESC control using PWM signals

## Hardware

* ESP32
* MPU6050
* 4 Brushless Motors
* 4 ESCs
* Li-Po Battery
* Quadcopter Frame

## Libraries

* ESP32Servo
* MPU6050 MotionApps
* I2Cdev
* SimpleKalmanFilter

## Project Status

⚠️ This project is still under development.

* PID parameters are not fully tuned.
* Flight stability is not yet reliable.
* Current version is intended for testing and demonstration purposes only.
